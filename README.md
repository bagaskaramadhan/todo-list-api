# Todo List API

## 1. Description
An efficient Task Management API (Todo List) built using RESTful architecture.
This project is designed to streamline daily activity management with a comprehensive set of CRUD (Create, Read, Update, Delete) features.
Powered by Express.js for seamless routing and MongoDB for flexible data storage.

## 2. Technologies
This project is built using the following technologies:
* **Language:** Node.js
* **Framework:** Express.js
* **Database:** MongoDB (NoSQL)
* **ODM:** Mongoose (Optional, but commonly used with MongoDB)

## 3. Features
The API provides the following endpoints to manage your tasks:

* **Get All Tasks**: Retrieve a complete list of all stored tasks.
* **Get Task by ID**: Fetch specific details of a single task using its unique ID.
* **Create Task**: Add new tasks to the database.
* **Update Task**: Modify existing task information (e.g., changing title or status).
* **Delete Task**: Remove tasks from the list.
